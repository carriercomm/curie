Incoming email processing

This is the email processing pipeline:

    Maildir
       ↓
   watcher.py       // Watcher catches changes in Maildir, moves files
       |            // to a storage dir and pushes messages to "inbox" queue.
       |            // Message is just a string - a path to a raw email blob.
       ↓
+--------------+
|    inbox     |
+------+-------+
       ↓
    parser.sh       // Parser (java code) parses email messages,
       |            // creates *.raw.json and *.parsed.json files
       |            // (according to schemas) next to a raw email file
       ↓
+--------------+
|    parsed    |
+------+-------+
       ↓
   indexer.py       // Reads *.parsed.json files and pushes
       |            // updates to a Solr instance
       ↓
+--------------+
|   indexed    |    
+------+-------+
       ↓
   labeler.py       // Reads pre-defined filters (global and
       |            // user specific) and adds
       |            // labels to a doc in Solr index 
       ↓
+--------------+
|   labeled    |
+------+-------+
